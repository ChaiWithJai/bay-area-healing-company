## Synthesis: Design Decisions for Bay Area Healing Company

### Color Palette (Inspired by templates)
| Token | Value | Rationale |
|-------|-------|-----------|
| Background | `#FAF8F3` | Warm off-white (Wellbe + Healing Quest inspired) |
| Primary | `#1C3D3D` | Deep teal (Healers dark sections) |
| Secondary | `#86A789` | Sage green (common across templates) |
| Accent | `#E8B86D` | Warm gold (Healing Quest inspired) |
| Text | `#2D3436` | Dark gray for readability |

### Typography
| Element | Choice | Rationale |
|---------|--------|-----------|
| Display | Fraunces or Playfair Display | Elegant serif like Cormorant/Abril Fatface |
| Body | Inter or DM Sans | Clean professional sans-serif |
| Accent | DM Mono | For "competent professional" code/data feel |

### Structure (Sections)
1. **Hero** - Bold headline, coach photo, clear CTA
2. **Programs** - Workshops, Courses, Cohorts, Events/Retreats
3. **Process** - How it works (numbered steps)
4. **About/Team** - Founder/coaches with credentials
5. **Testimonials** - Real results, professional language
6. **FAQ** - Address common questions
7. **CTA** - Book a call / Start journey
8. **Footer** - Contact, social, newsletter

### Key Design Principles
1. **Goal-oriented, not woo-woo** - Lead with outcomes, not generic wellness speak
2. **Competent user respect** - Dense info, no hand-holding
3. **Professional but warm** - Elegant typography, warm colors
4. **Clear pricing/structure** - Programs clearly defined
5. **Social proof** - Testimonials from professionals
6. **Process-focused** - Show the journey/steps

### Technical Approach
- Single-page HTML with CSS custom properties
- Smooth scroll navigation
- CSS animations for reveals
- Mobile-responsive (mobile-first)
- Fast loading (no heavy frameworks)
- Deploy to Netlify
